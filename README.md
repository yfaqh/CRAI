## 🚩  项目简介

**伴读AI**项目以人机交互为核心，聚焦于探索人工智能新技术在现代化阅读中的应用，并基于计算机视觉技术，研究阅读场景中的**人机交互**与**智能辅助阅读**技术。项目包含***手-文混合目标检测***、***场景文本图像超分辨率***等多个方面的算法创新，基于所创新的算法实现了流畅与准确的辅助阅读，提高了阅读学习效率。项目进一步通过开发移动端平台，提供复习与二次学习等个性化功能。
